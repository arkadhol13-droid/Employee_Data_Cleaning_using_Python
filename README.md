# Indian Employee Data Cleaning using Python

## Project Overview
This project focuses on cleaning and preprocessing an employee dataset using Python.  
The goal is to prepare raw data for further analysis by removing inconsistencies and handling missing values.

## Dataset
The dataset contains employee information such as:

- Employee details
- Salary (INR)
- Performance Rating
- Other employee attributes

## Data Cleaning Steps

The following data cleaning operations were performed:

- Handling missing values
- Converting salary values to numeric format
- Removing duplicate records
- Replacing negative salary values
- Treating infinite values
- Handling outliers using statistical methods
- Filling missing numeric values with mean/median

## Tools & Libraries

The project uses the following Python libraries:

- Python
- Pandas
- NumPy

## Project Workflow

1. Load dataset using Pandas
2. Check missing values
3. Convert salary and rating columns to numeric
4. Handle missing values
5. Remove duplicate records
6. Fix negative salary values
7. Remove outliers
8. Export cleaned dataset
